Burger List

With this app you can save information, move information, and delete information from a database.
You can add a burger to either an 'eaten' or 'not eaten' list. You can switch added burgers between both lists. You can also delete a burger from either list.

Built with:

  • JavaScript
  • MySQL
  • Handlebars
  • Node.js
    • express - for local serve
    • sequelize - promise-based Node.js ORM for MySQL
    • fs & path - useful functionality to access and interact with the file system
